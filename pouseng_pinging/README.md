# 莆仙話拼音輸入方案 Pouseng Ping'ing Input Method

莆仙話拼音（Pouseng Ping'ing），又稱興化拼音、興拼，是現代莆仙語圈子最常用的羅馬字拼音方案，能準確標記多種莆仙語方言。

## 設計理念

莆仙話拼音採用折衷設計：

- **聲母**：採用類普拼式（b/p/d/t/z/c），方便初學者學習
- **韻母**：採用國際音標轉寫式，更為精確
- **聲調**：使用數字標記（1-7），清晰易辨

這種設計既保持了音韻的準確性，又降低了學習門檻。

## 輸入方案特色

### 1. 無聲調輸入

**輸入時不需要輸入聲調**，輸入法會自動列出所有可能的候選詞：

```
輸入：putian    → 候選：莆田 pou²leng²
輸入：hinghwa   → 候選：興化 hing¹ngua⁴
```

候選詞會顯示聲調數字以便辨識。

### 2. 簡拼支援

支援首字母縮寫快速輸入：

```
輸入：pt       → 候選：莆田
輸入：hh       → 候選：興化
輸入：b        → 候選：白、巴、爸、...
```

### 3. 模糊音支援

#### 類普拼式（方便初學者）

對不熟悉莆仙話拼音的使用者，可用類似漢語拼音的寫法：

| 正確寫法 | 類普拼寫法 | 範例 |
|---------|-----------|------|
| y | v | 魚 hv → hy |
| au | ao | 豆 dao → dau |
| ieng | ian, yan | 咸 gian → gieng |
| ieu | iao | 藥 iao → ieu |
| yorng | iong | 傷 siong → syorng |
| ia | ya | 野 ya → ia |
| i | yi | 衣 yi → i |
| iu | you | 游 you → iu |
| y | yu | 於 yu → y |
| uang | wan | 彎 wan → uang |
| ue | wei | 丸 wei → ue |
| ui | wui | 為 wui → ui |
| ua | wa | 畫 wa → ua |
| u | wu | 污 wu → u |

#### 簡碼輸入

某些複雜韻母可用簡化寫法：

| 完整寫法 | 簡碼 | 範例 |
|---------|------|------|
| yor | yo, io | 蟻 hyo, hio → hyor |
| yorng | yong, ion | 強 gyong → gyorng |
| yorh | yoh | 約 yoh → yorh |
| orh | oq | 國 goq → gorh |
| orng | om | 工 gom → gorng |
| zyor | jo | 紙 jo → zyor |
| zyorng | jong, jon | 章 jong → zyorng |
| ung | ng | 囥 kng → kung |

#### 鼻音韻尾簡化

可省略 -ng 的 g：

| 完整寫法 | 簡化寫法 | 範例 |
|---------|---------|------|
| ang | an | 安 an → ang |
| eng | en | 邊 ben → beng |
| ing | in | 冰 bin → bing |
| ong | on | 分 bon → bong |
| ung | un | 軍 gun → gung |

#### 顎化音

z/c/s + i/y 可寫成 j/q/x + i/u：

| 原寫法 | 顎化寫法 | 範例 |
|-------|---------|------|
| zi | ji | 字 ji → zi |
| ci | qi | 飼 qi → ci |
| si | xi | 舌 xi → si |
| zy | ju | 煮 ju → zy |
| cy | qu | 鼠 qu → cy |
| sy | xu | 薯 xu → sy |

#### 入聲韻尾模糊

入聲韻尾 -h 和 -k 可互換或省略：

```
輸入：ba       → 候選：白 bah⁶、巴 ba¹
輸入：bak      → 候選：白 bah⁶
輸入：bah      → 候選：白 bah⁶
```

## 變調與連讀

莆仙話有複雜的變調和聲母類化現象：

### 變調規則

在多字詞中，**只有最後一個字保持本調，其他字都會變調**。

例如：
- 「莆田」：pou² + ceng² → pou⁵ leng²（第一字變調，第二字類化）
- 「興化語」：hing¹ + ngua⁴ + gy³ → hing⁵ nua⁴ gy³

### 聲母類化

非詞首音節的聲母會受前一音節韻尾影響而變化：

**規則一：前字以 -ng 結尾**
- b, p, m → m
- d, t, n, l, z, c, s → n
- g, k, h, ng, (零) → ng

**規則二：前字以 -h 結尾**
- 聲母不變

**規則三：前字為開音節**
- b, p → (零) 或 w
- d, t, z, c, s → l
- g, k, h, (零) → (零)
- m, n, l, ng → 保持不變

**注意**：詞庫中記錄的是**本調和原始聲母**，輸入時可用任何形式（本調、變調、類化、原始）進行檢索。
